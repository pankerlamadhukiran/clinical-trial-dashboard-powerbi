# 🧪 Clinical Trial Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an end-to-end clinical trial analytics dashboard developed using Power BI and Excel. The dashboard simulates a real-world clinical study and provides insights into treatment efficacy, safety outcomes, and biomarker changes across different study arms.

---

## 🎯 Objectives
- Design a structured clinical dataset
- Perform data cleaning and transformation
- Build a relational data model using Patient_ID
- Analyze treatment effectiveness and safety
- Create an interactive dashboard for decision-making

---

## 📊 Dataset Description
The dataset is organized into multiple related tables:

- **Demographics**: Age, Gender, Study Arm, Weight  
- **Vitals**: Blood Pressure (Baseline, Week4, Week8, Week12)  
- **Lab Results**: Hemoglobin, WBC, Platelets  
- **Biomarkers**: CRP (mg/L), IL-6 (pg/mL)  
- **Safety Outcomes**: Adverse Events (Mild, Moderate, Severe)  
- **Efficacy Outcomes**: Responder, Partial Responder, Non-Responder  

All tables are connected using a common **Patient_ID**, enabling relational data modeling.

## 🔍 Key Insights
- High Dose group showed better response rates compared to Placebo  
- Significant reduction in blood pressure observed over time  
- CRP and IL-6 levels decreased in treatment groups  
- Higher dose groups showed slightly increased adverse events
